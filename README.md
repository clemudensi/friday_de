# Getting Started

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Steps to run application.
1. Step 1 (api-server)
   * cd `apiserver`
   * run yarn
   * yarn start
2. Step 2 (frontend app)
   * cd `car_maker_frontend`
   * run yarn
   * yarn start
